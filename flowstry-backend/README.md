# Flowstry Backend

The core backend service for Flowstry, written in Go.

> **Detailed Documentation**: Please see [docs/services/backend.md](../docs/services/backend.md) for architecture, configuration, and API details.


## Getting Started

### Prerequisites
- Go 1.21+

### Installation

```bash
cd flowstry-backend
go mod download
```

### Development

```bash
go run main.go
# or via air if configured
```

## Configuration

Environment variables can be set via `.env` file or exported in the shell.
