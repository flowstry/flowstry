# Flowstry Website

The **Flowstry Website** is the public-facing marketing site.

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Firebase Hosting

## Purpose
- Landing page for product introduction.
- Documentation/Blog (if applicable).
- Login/Signup entry points.

## Development

```bash
# Start development server
npm run dev

# Build
npm run build
```
