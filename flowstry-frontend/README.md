# Flowstry Frontend

The main frontend application for the Flowstry platform, built with [Next.js](https://nextjs.org/).

## Getting Started

### Prerequisites
- Node.js 18+
- npm/yarn/pnpm

### Installation

```bash
cd flowstry-frontend
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

## Build

```bash
npm run build
```
